# Multilevel Audit

Check:

1. The theorized level: individual, dyadic, leader, team, organization, or cross-level.
2. Actual nesting units and cluster counts.
3. ICC values and aggregation evidence where relevant.
4. Whether standard errors account for clustering.
5. Random intercepts, random slopes, fixed effects, and centering choices.
6. Whether group-mean centering changes the interpretation from a contextual attribute to a within-group deviation.
7. Whether CFA also accounts for clustering when needed.
